# 📚 Estante Virtual

Projeto desenvolvido em Java utilizando JDBC para gerenciamento de uma estante virtual.

## 📌 Funcionalidades

- Cadastro de Livros
- Cadastro de Autores
- Cadastro de Editoras
- Operações CRUD:
  - Inserir
  - Alterar
  - Excluir
  - Consultar
  - Listar

## 🛠️ Tecnologias utilizadas

- Java
- PostgreSQL
- NetBeans

## 📂 Estrutura do Projeto

- `model` → classes com os dados (Livro, Autor, Editora)
- `persistencia` → acesso ao banco de dados
- `negocio` → regras do sistema
- `util` → conexão com banco e utilidades
- `test` → testes do sistema
